# 🎓 Student Grading System

A simple Windows Forms application developed using **VB.NET** for computing and managing students' grades.

## 📌 Features

- Add up to 6 student names.
- Select Course.
- Select Year & Section.
- Select Semester.
- Compute Final Grades.
- Automatic Pass/Fail Status.
- Input Validation:
  - Student Name Required
  - Numbers Only
  - Maximum Grade is 100
  - Required Field Validation
- Save Grade Records.

## 🖥️ Built With

- Visual Basic .NET (VB.NET)
- Windows Forms

## 🚀 How to Run

1. Open the project in **Visual Studio 2022**.
2. Open `Grading_System.slnx`.
3. Press **F5** or click **Start**.
4. Enter student information.
5. Click **Compute** to calculate grades.
6. Click **Save** to save the record.

## 📖 Validation Rules

- Student Name is required.
- Grades must contain numbers only.
- Maximum grade allowed is **100**.
- Empty fields are not accepted.
